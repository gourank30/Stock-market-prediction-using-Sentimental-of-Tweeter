#stockmarket with sentiment analysis

Python version is 3.6.3 is required with windows 10

1.For generating stock dataset run yahoo.py
	Libraries used for fetching stock data are
		1. yfinance
		2. datetime

2.For generating visualizations of the stock market data run visualization.py	
	Libraries used for visualization of data are
		1. matplotlib
		2. pandas
		3. numpy

3.For fetching twitter tweets of stock run tweetData.py
	Libraries used for fetching tweets data are:
		1. tweepy
		2. pandas
		3. numpy
		
4.Traing and Testing run soln.py
	Libraries need to be installed are
	1. tensorflow
	2. keras
	3. pandas
	4. numpy
	5. textblob (sentiment analysis)
	6. tkinter (gui)
